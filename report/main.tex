\begin{abstract}
This report presents an end-to-end AI system that automates product detection from retail shelf
images, maintains a continuously updated inventory database, and exposes a natural-language
business-intelligence interface. The detection backbone is YOLOv8m fine-tuned on the SKU-110K
dataset; classification combines a ResNet-50 coarse classifier with CLIP-based fine-grained
retrieval; unknown products are auto-labeled via LLaVA; and a LangGraph/Ollama pipeline answers
plain-English inventory queries over a guarded, read-only SQL interface.
